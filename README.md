# BlackJack

A browser-based Blackjack card game built with Next.js.

🌐 **Live:** [blackjack.paulkuehn.ch](https://blackjack.paulkuehn.ch)

---

## Stack

Next.js 15 · React 19 · CSS · Vercel

---

## Features

- Full Blackjack gameplay with betting, hit, stand, double down, and split
- Animated card dealing with timed sequences
- Keyboard controls — `W` Hit · `S` Stand · `D` Double · `A` Split
- Split hands with independent bets and double after split
- Basic strategy training mode
- Pro mode with card counting *(coming soon)*

---

## Rules

- Dealer stands on soft 17
- No surrender
- No re-split after split

---

## Getting Started

```bash
git clone https://github.com/endodod/BlackJack.git
cd BlackJack
npm install
npm run dev
```
