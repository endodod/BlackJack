# Sound Assets

Drop your sound files here to override the default generated sounds.

**Expected file names (used by the app):**
- `shuffle.mp3` (shuffle sound)
- `deal.mp3` (deal sound)
- `draw.mp3` (card draw / hit sound)
- `win.mp3` (win / blackjack sound)
- `bust.mp3` (bust sound)
- `push.mp3` (push sound)

Once the file is added, the app will automatically use it for the corresponding sound effect.

Supported file formats depend on the browser (e.g., .mp3, .wav, .ogg). If a file is missing, the app falls back to the built-in generated sound.
